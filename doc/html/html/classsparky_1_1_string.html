Detailed Description

sparky::String is a basic re-implementation of C++ string class specifically made for easily functionality with Sparky. A sparky::String is container for a vector of characters that can be easily manipulated without dynamic allocation.

sparky::String contains several methods for simple manipulation of characters; such as concatenation, easy deletion and per index alterations. Below is an example:

Usage example:

// Declare a string
sparky::String str = "Hello";
// Append another string onto str
str.append(" World!");
// Store a substring of str
sparky::String sub = str.substring(0, 5);
// Delete the l letters in str
str.remove("l");
// Print the strings to the console
std::cout << str << std::endl;
std::cout << sub << std::endl;
