Detailed Description

sparky::PoolManager is a singleton class that is responsible for the basic garbage collection of Sparky. Dynamic Ref objects are added to the list. At the end of the current frame, each objects reference count is decremented and if the count is 0, it is deleted.

Destructs the PoolManager object instance.

The flush method is called once more upon the objects destruction to make sure that it is not retaining any objects, therefore preventing memory leaks.
