Detailed Description

sparky::Light is the base parent class for all types of lighting within the Sparky Engine. It controls the position and overall colour of the light within the scene. The light itself cannot be used and can only be utilised through it's child classes. A code example is provided in sparky::PointLight and sparky::DirectionalLight.
