Detailed Description

sparky::World is the culmination of all the Chunk's and Voxels within the application in one class. The World controls the rendering and initialisation of Chunk objects.

Chunks can be added and removed from the World and edited through various convenience methods. Below is a code example of using the World.

Usage example:

// Create a new World and retain!
sparky::World* pWorld = new sparky::World();
pWorld->addRef();
// Add a Chunk to the World.
pWorld->addChunk(sparky::Vector3i::zero());
// Render the World!
pWorld->render(sparky::ResourceManager::getInstance().getShader("deferred"));
