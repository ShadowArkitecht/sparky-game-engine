Detailed Description

The Program class is responsible for the linking and joint compilation of all attached shaders. The Program class can also bind and unbind this shader functionality to the currently rendering objects.

It provides an easy-to-use interface for shader functionality, without the user having to change various states of the engine. Below is a code example of using the Program.

Usage example:

// Make a program and add shaders.
sparky::Program program;
program.attachShader(new sparky::GLSLObject("shaders/basic_vertex.glsl", sparky::eShaderType::VERTEX_SHADER));
program.attachShader(new sparky::GLSLObject("shaders/basic_fragment.glsl", sparky::eShaderType::FRAGMENT_SHADER));
// Link and compile the shaders.
program.link();
// Bind the shader for use.
program.bind();
