Detailed Description

sparky::Scene is an abstract class in which all scenes will inherit from in an application. A scene refers to a level in a game, the scenes can be pushed and popped from the Game Manager stack so that more functionality can easily be added or removed.

Below is a code example where a child class for Game has theoretically been created.

Usage example:

// Create a new Game object.
Game* pGame = new Game();
// Add the game as the first level in the application.
sparky::GameManager::getInstance().pushScene(pGame);
