Detailed Description

sparky::GLSLObject is a OpenGL Shader wrapper class for easily loading, parsing and compiling GLSL shaders. It provides additional error checking and helper methods for making compilation as simple as possible.

sparky::GLSLObject is often used in conjunction with sparky::Program, which provides additional functionality for Shader objects.

Due to the generic nature of the class, it can be used for compilation of any shader that the user provides it with. For a code example please refer to sparky::Program.
