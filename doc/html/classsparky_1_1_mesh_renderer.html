Constructs a MeshRenderer object with a Mesh and Texture.

The MeshRenderer is a convenience class for rendering a Mesh and applying a texture to it. It is a renderable component so it can be attached to the GameObject for rendering.

Parameters
pMeshThe Mesh of the renderer.
pTextureThe Texture to be applied to the Mesh.
